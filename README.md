### 🚀 O que ando construindo

Meu repositório é um laboratório. Aqui estão alguns dos projetos que decidi deixar publico:

*   [**Gateway-PIX-Autonomo**](https://github.com/CillsGhost2/Gateway-PIX-Autonomo): Um gateway de pagamentos de baixo custo usando Netlify Functions e Mercado Pago. Ele simplesmente verifica se o pagamento caiu na conta (ps: usei scrape e não a API padrão e bem documentada)
*   [**Hotmart-Account-Checker**](https://github.com/CillsGhost2/Hotmart-Account-Checker): Um checker de logins + extrator de cursos de contas da Hotmart. Muito bom pra caso tenha que controlar contas de afiliados e afins. Feito usando Playwright no Python.

### 📂 Outros projetos

*   **Mobile & Push Notifications**: App de agendamentos desenvolvido do zero com Flutter, Dart e integração com Firebase/OneSignal. Pro meu TCC.
*   **Experimentos Serverless**: Projetos como *FakeBet* (https://fakebet.neocities.org/) e chats (https://cillsghost.neocities.org/) hospedados no Neocities, utilizando Firebase Realtime Database para gerenciar estados complexos diretamente pelo cliente. Partes de um desafio proposto por mim para contornar o CSP do Neocities e erros de CORS do navegador.
